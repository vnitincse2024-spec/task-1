# Portfolio Website - Task 1

## Project Description
A responsive personal portfolio website built with HTML, CSS, and modern web development practices. This project demonstrates competency in front-end web development, responsive design, and version control with Git/GitHub.

## Features
- ✅ **3 Responsive Pages**: Home, About, and Contact
- ✅ **Modern CSS**: Flexbox and Grid layout with gradient backgrounds
- ✅ **Responsive Design**: Mobile-first approach with media queries
- ✅ **Professional Navigation**: Consistent navigation bar across all pages
- ✅ **Interactive Elements**: Hover effects and smooth transitions
- ✅ **Contact Form**: Functional contact form with modern styling

## File Structure
```
task-1/
├── index.html       # Home page with hero section and skills showcase
├── about.html       # About page with education and experience details
├── contact.html     # Contact page with form and contact information
├── styles/
│   └── main.css     # Main stylesheet with responsive design
└── README.md        # Project documentation
```

## Technologies Used
- **HTML5**: Semantic markup and modern structure
- **CSS3**: Flexbox, Grid, gradients, transitions, and animations
- **Responsive Design**: Media queries for mobile (≤768px) and desktop
- **Git & GitHub**: Version control and collaboration

## Setup Instructions

### Local Development
1. Clone this repository:
   ```bash
   git clone https://github.com/vnitincse2024-spec/task-1.git
   ```

2. Navigate to the project directory:
   ```bash
   cd task-1
   ```

3. Open `index.html` in your browser or use a live server (recommended):
   - **VS Code**: Install Live Server extension and click "Go Live"
   - **Python**: Run `python -m http.server 8000`
   - **Node.js**: Run `npx http-server`

### Deployment
This site is deployed via **GitHub Pages**.

**Live URL**: `https://vnitincse2024-spec.github.io/task-1/`

## Design Choices

### Color Scheme
- **Primary Gradient**: Purple to violet (`#667eea` to `#764ba2`)
- **Background**: Light gray (`#f4f4f4`)
- **Text**: Dark gray (`#333`)
- **Accent**: White for buttons and highlights

### Layout
- **Navigation**: Flexbox for responsive navigation bar
- **Skills Grid**: CSS Grid with auto-fit for flexible card layout
- **Contact Form**: Grid layout for side-by-side form and info on desktop

### Responsive Breakpoint
- **Desktop**: Default styles (>768px)
- **Mobile**: Stacked layout, adjusted font sizes (≤768px)
